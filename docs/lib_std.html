itobcd
  • Input(s): Bemenet(ek): R0=unsigned binary value (0-99999999) R0=bináris szám (0-99999999)
  • Output(s): Kimenet(ek): R4=BCD coded value R4=BCD kódú szám

Előjel nélküli bináris kódú szám átalakítása BCD kódolásra. A bemeneti érték maximum 99999999 lehet, egyébként az eredmény 0 lesz.
